## 目标

komari-theme-emerald 原始项目：https://github.com/Tokinx/komari-theme-emerald

计划将原始 komari-theme-emerald 主题移植到 CF Server Monitor。

基于 Vue 3 + Vite + reka-ui + Tailwind CSS v4 构建（移除 Cobe Earth）
